Editing/Creation Menu

Create New ISTC Entry

This option allows you to create a new ISTC entry from scratch. You are not able to create a new ISTC entry with the same ISTC Number as one in the database or one being created by yourself or another user.

Edit Existing ISTC Entry

This option allow you to edit a record either from the main database or from your draft file store.

Delete Existing ISTC Entry

This option allows you to delete an ISTC entry already in the database.

Add/Edit Bibliographical Reference Abbreviations

This option allows you to edit and add Bibliographical Reference Abbreviations to use in the ISTC records. Abbreviations can also be edited and added while creating or editing an ISTC record.

Add/Edit USA Location Abbreviations

This option allows you to edit and add USA Locations to use in the ISTC records. USA locations can also be edited and added while creating or editing an ISTC record.

Edit Your User Account

The option allows you to edit your user account and change your password.
